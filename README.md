# LarkTeX — Конвертер русского математического текста в LaTeX

[![Python 3.12+](https://img.shields.io/badge/python-3.12+-blue.svg)](https://www.python.org/downloads/)

Преобразует математический текст на русском языке в формат LaTeX.

```
"синус икс плюс корень из а" → "\sin(x) + \sqrt{a}"
```

## Быстрый старт

```bash
# Установка
uv sync

# Запуск
uv run larktex.py "синус икс"
# Вывод: \sin(x)

# Тесты
uv run pytest tests/ -v
```

## Примеры

| Русский текст | LaTeX |
|---------------|-------|
| `синус икс` | `\sin(x)` |
| `икс в квадрате` | `x^2` |
| `дробь один на два` | `\frac{1}{2}` |
| `ц из 5 по 2` | `\binom{5}{2}` |
| `эн факториал` | `n!` |
| `производная от икс по иксу` | `\frac{d}{dx}` |
| `предел икс стремится к бесконечности` | `\lim_{x \to inf}` |

## Поддерживаемые конструкции

- **Функции:** sin, cos, tan, cot, arcsin, arccos, exp, log, ln, sqrt
- **Арифметика:** +, -, *, /, ^, скобки
- **Пределы:** `предел икс стремится к нулю`
- **Интегралы:** `интеграл от икс де икс`
- **Суммы/произведения:** `сумма от ... по ...`
- **Производные:** `производная`, `вторую производную`, `частная`
- **Дроби:** `дробь ... на ...`
- **Факториалы:** `эн факториал`, `5 двойной факториал`
- **Биномы:** `ц из n по k`, `а из n по k`

## Архитектура

```
Русский текст → normalizer.py → grammar.lark → parser.py → LaTeX
```
