# Math Text to LaTeX Parser

## Архитектура

Проект преобразует русский математический текст в LaTeX через три этапа:

```
Русский текст → normalizer.py → grammar.lark → parser.py → LaTeX
```

## Компоненты

### normalizer.py
Нормализатор переводит русский математический текст в латиницу.

**Основные функции:**
- Загрузка словарей из `i18n/`
- Обработка падежных форм
- Замена греческих букв на символы Unicode
- Преобразование степеней ("в квадрате" → "^2")
- Обработка дифференциала ("де" → "d")

**Примеры:**
- "синус от икс" → "sin x"
- "икс в квадрате" → "x^2"
- "альфа" → "α"

### grammar.lark
Описание грамматики для Lark парсера.

**Поддерживаемые конструкции:**
- Арифметика: `+`, `-`, `*`, `/`, `^`
- Функции: `sin`, `cos`, `tan`, `sqrt`, `log`, `ln` и др.
- Сравнения: `=`, `!=`, `<`, `>`, `<=`, `>=`, `≈`, `∼`
- Интегралы: `integral from a to b of f(x) d x`
- Суммы и произведения: `sum`, `product`
- Производные: `d x`

### parser.py
Transformer, преобразующий AST Lark в LaTeX код.

**Основные методы:**
- `parse(text: str) → str` — преобразует нормализованный текст в LaTeX
- `_add_parens(text)` — добавляет скобки к функциям без скобок


## Использование

```python
from normalizer import Normalizer
from parser import Parser

# Создание объектов
norm = Normalizer('i18n')
parser = Parser()

# Преобразование русского текста в LaTeX
text = "синус от икс плюс корень из а"
normed = norm.process(text)      # "sin x + sqrt a"
latex = parser.parse(normed)     # "\\sin(x) + \\sqrt{a}"

print(f"{text} → {latex}")
```

## Поддерживаемые конструкции

### Функции
| Русский | Нормализация | LaTeX |
|---------|--------------|-------|
| синус | sin(x) | \sin(x) |
| косинус | cos(x) | \cos(x) |
| тангенс | tan(x) | \tan(x) |
| корень | sqrt(a) | \sqrt{a} |
| логарифм | log(x) | \log(x) |

### Операторы
| Русский | Нормализация | LaTeX |
|---------|--------------|-------|
| равно | = | = |
| примерно | approx | \approx |
| плюс | + | + |
| умножить | * | \cdot |
| в квадрате | ^2 | ^2 |

### Греческие буквы
| Русский | Символ |
|---------|--------|
| альфа | α |
| бета | β |
| гамма | γ |
| дельта | δ |
| мю | μ |

## Ограничения

1. **Интегралы с пределами** — частично работают
2. **Кириллические буквы в переменных** — не заменяются
3. **Вложенные функции без скобок** — требуют явных скобок
