### Корни (в разработке)
| Термин | Русский | Английский | Статус |
|--------|---------|------------|--------|
| Квадратный корень | `корень из а` | `sqrt a` | 🔄 `sqrt{a}` |
| Кубический корень | `корень кубический из а` | `cbrt a` | 🔄 `sqrt[3]{a}` |
| Корень степени N | `корень степени 5 из а` | `sqrt[5]{a}` | 🔄 `sqrt[5]{a}` |
| Корень N степени | `корень 5 степени из а` | `sqrt[5]{a}` | 🔄 `sqrt[5]{a}` |
| X корень степени N | `икс корень степени 5` | `x sqrt[5]{...}` | ❌ Требуется контекст |

**Правила нормализации корней:**
1. **Квадратный корень:** `корень из X` → `sqrt{X}`
2. **Кубический корень:** `корень кубический из X` → `sqrt[3]{X}`
3. **Корень степени N:** `корень степени N из X` → `sqrt[N]{X}`
4. **Корень N степени:** `корень N степени из X` → `sqrt[N]{X}`
5. **Границы выражения:**
   - *выражение_степени* ограничено справа: `all`/`все`/`всё`, или предлогом `от`/`из`
   - *подкоренное_выражение* ограничено справа: `all`/`все`/`всё`, или концом выражения

### Степени (в разработке)
| Термин | Русский | Английский | Статус |
|--------|---------|------------|--------|
| В квадрате | `икс в квадрате` | `x squared` | 🔄 `x^{2}` |
| В кубе | `икс в кубе` | `x cubed` | 🔄 `x^{3}` |
| В N степени | `икс в степени 5` | `x to the 5` | 🔄 `x^{5}` |
| В N-й степени | `икс в пятой` | `x to the 5th` | 🔄 `x^{5}` |
| В выражение степени | `икс в степени три минус пять` | `x^{3-5}` | 🔄 `x^{3-5}` |

**Правила нормализации степеней:**
1. **Форматы:**
   - `в квадрате` → `^2`
   - `в кубе` → `^3`
   - `в степени N` → `^{N}`
   - `в N степени` → `^{N}`
   - `в N-й` (пятая, шестая и т.д.) → `^{N}`

2. **Границы выражения степени:**
   - *выражение_степени* ограничено справа: `all`/`все`/`всё`, или концом выражения
   - НЕТ предлогов `от`/`из` внутри степени

**Примеры:**
- `x в степени три минус пять плюс два плюс пять` → `x^{3} - 5 + 2 + 5`
- `x в степени три минус пять плюс два всё плюс пять` → `x^{3-5+2} + 5`
- `a в пятой плюс икс плюс игрек все плюс зет` → `a^{5 + x + y} + z`
