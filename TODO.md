### Математические функции
- [ ] `arccos`, `arctan`, `arccot`
- [ ] `arcsinh`, `arccosh`, `arctanh` (обратные гиперболические)
- [ ] `coth`, `sech`, `csch` (гиперболические котангенс и др.)
- [ ] `sec`, `csc` (секанс и косеканс)

- [ ] `abs` (модуль/абсолютное значение)
- [ ] `exp` (экспонента)
- [ ] `ln` (натуральный логарифм)

### Операторы сравнения
- [ ] `≠` (не равно) — `neq`
- [ ] `≤` (меньше или равно) — `leq`
- [ ] `≥` (больше или равно) — `geq`
- [ ] `∼` (подобно/асимптотически) — `sim`
- [ ] `≡` (тождественно равно) — `equiv`
- [ ] `∝` (пропорционально) — `propto`

### Матрицы и определители
- [ ] `матрица` — `\begin{matrix} ... \end{matrix}`
- [ ] `определитель` — `\det`
- [ ] `транспонированная` — `^T`

### Дроби
- [ ] `дробь из а на б` → `\frac{a}{b}`
- [ ] `а делить на б` → `a / b`

### Производные
- [ ] `производная от икс` → `\frac{d}{dx}`
- [ ] `вторую производную` → `\frac{d^2}{dx^2}`
- [ ] `частную производную` → `\partial`

### Суммы и произведения
- [ ] `сумма от ноль до бесконечности` → `\sum_{n=0}^{\infty}`
- [ ] `произведение от один до n` → `\prod_{i=1}^{n}`

### Интегралы (приоритет)
- [ ] `интеграл от нуля до единицы` — исправить текущую реализацию
- [ ] `кратный интеграл` — `\iint`, `\iiint`
- [ ] `криволинейный интеграл` — `\oint`

### Специальные символы
- [ ] `бесконечность` → `\infty`
- [ ] `пустое множество` → `\emptyset`
- [ ] `принадлежит` → `\in`
- [ ] `не принадлежит` → `\notin`
- [ ] `подмножество` → `\subset`
- [ ] `объединение` → `\cup`
- [ ] `пересечение` → `\cap`

2. **Матанализ** — пределы, производные, интегралы

4. **Тригонометрия** — тождества, уравнения
