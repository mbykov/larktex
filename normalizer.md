# Нормализатор математического текста

## Описание

Нормализатор преобразует русский текст с математическими выражениями в латиницу. Это промежуточный этап перед парсингом в LaTeX.

## Архитектура

```
Русский текст → normalizer.py → Латиница → grammar.lark → parser.py → LaTeX
```

## Основные шаги обработки

### 1. Загрузка словарей

При инициализации нормализатор загружает два JSON-файла из папки `i18n/`:

- **synonyms_ru.json** — словарь синонимов с падежными формами
  - Переменные (икс, игрек, зет...)
  - Греческие буквы (альфа, бета, гамма...)
  - Функции (синус, косинус, тангенс...)
  - Операторы (плюс, минус, умножить, делить...)
  - Степени (в квадрате, в кубе...)
  - Числа (ноль, один, два...)

- **ru.json** — метаданные языка
  - Служебные слова (noise_words)
  - Стоп-слова (stop_words)
  - Союзы (conjunctions)

### 2. Построение обратного отображения

Создаётся словарь для быстрого поиска: каждое слово → его нормальная форма.

Пример:
- "икс" → "x"
- "синус" → "sin"
- "альфа" → "α"

### 3. Обработка степеней (первый проход)

Сначала заменяются фразы со "в":
- "в квадрате" → "^2"
- "в кубе" → "^3"

Это делается до удаления предлога "в".

### 4. Удаление предлогов

Удаляются математически несущественные предлоги:
- "от"
- "из"
- "на"
- "до"

Пример: "интеграл от икс до игрек" → "integral икс игрек"

### 5. Обработка дифференциала

"де" заменяется на "d" (дифференциал):
- "де икс" → "d x"

Важно: "дэ" (буква D) не заменяется на "d", чтобы не конфликтовать.

### 6. Обработка скобок

- "открыть скобку" → "("
- "закрыть скобку" → ")"

### 7. Замена синонимов на латиницу

Основной этап. Слова заменяются по словарю в порядке убывания длины (сначала длинные фразы):

1. Сортируем все синонимы по длине (от длинных к коротким)
2. Для каждого синонима ищем в тексте и заменяем на нормальную форму
3. Пропускаем уже обработанные слова ("от", "из", "де", "в" и т.д.)
4. Пропускаем однобуквенные латинские символы (чтобы не менять x → X)

Примеры:
- "синус" → "sin"
- "икс" → "x"
- "альфа" → "α"
- "дэ заглавная" → "D"

### 8. Очистка пробелов

- Несколько пробелов → один
- Пробел перед "^" удаляется ("x ^2" → "x^2")
- Пробел после "(" удаляется ("( x" → "(x")
- Пробел перед ")" удаляется ("y )" → "y)")

## Примеры работы

| Вход | Выход |
|------|-------|
| "синус от икс" | "sin x" |
| "икс равно игрек" | "x = y" |
| "корень из а плюс б" | "sqrt a + b" |
| "икс в квадрате" | "x^2" |
| "интеграл синус икс де икс" | "integral sin x d x" |
| "дэ заглавная" | "D" |
| "мю равно хи" | "μ = χ" |
| "тэ заглавная приблизительно эй" | "T approx E" |

## Обработка заглавных букв

- **Строчные**: "икс" → "x", "ю" → "u"
- **Заглавные**: "икс заглавная" → "X", "ю заглавная" → "U"
- **Синонимы**: "оу" → "U", "эй" → "E", "би" → "B"

## Обработка греческих букв

Все греческие буквы заменяются на символы Unicode:
- "альфа" → "α"
- "бета" → "β"
- "дельта" → "δ"
- "мю" → "μ"
- "хи" → "χ"
- "эта/эт" → "η"

## Интерфейс

```python
from normalizer import Normalizer

normalizer = Normalizer(i18n_dir="i18n")

# Обработка одной строки
result = normalizer.process("синус от икс")
# Результат: "sin x"

# Обработка текста с выделением математических островов
islands = normalizer.extract_math_islands("пусть икс равно синус от у")
# Результат: [("пусть ", False), ("икс равно синус от у", True)]
```

## Ограничения

1. **Нормализатор не создаёт LaTeX** — только переводит в латиницу
2. **Нет проверки контекста** — "ю" всегда → "u" (строчная), "ю заглавная" → "U"
3. **Некоторые слова остаются** — "ипсилон", "ди" не всегда заменяются
4. **Падежи упрощены** — все падежные формы ведут к одной базовой

## Следующие этапы

После нормализации текст передаётся в:
1. **grammar.lark** — описание грамматики для Lark парсера
2. **parser.py** — Transformer, преобразующий AST в LaTeX
